常見問題

「ew」是什麼？這個伺服器是做什麼的？

「ew」是一個用 Rust 編寫的《Love Live! School idol festival 2 MIRACLE LIVE!》私服。這款遊戲是 Love Live! 系列題材的手機節奏遊戲。

我只是想連到這個伺服器遊玩，要怎麼安裝 App？（Android）

此伺服器管理員沒有提供預先修補好的 APK。請參考下一個問題。

伺服器管理員沒有提供下載連結，我要怎麼修補 APK？（Android）

你可以使用 @arasfon 的 sif2 apk patcher。把資源 URL「https://sif2.sif.moe」填入「Assets URL」，再填入伺服器 URL（通常是「」，但實際情況可能不同）。選擇遊戲版本，將「header format」設為「Lowercase」，然後按下 patch。完成後使用 uber-apk-signer 簽名 APK，再傳到手機安裝。

我要怎麼安裝 App？（iOS）

iOS 的安裝流程比 Android 簡單。先下載國際服或日服的 IPA，然後使用 Sideloadly 或你偏好的安裝工具側載 App。接著打開系統設定，找到剛安裝的 App，填入伺服器 URL（通常是「」，但實際情況可能不同）以及資源 URL「https://sif2.sif.moe」。如果 App 開啟時出錯，確認設定中的 URL 結尾不要帶斜線（/）。

救命！我打開 App 時顯示「不可用」（iOS）

不要刪除 App，重新側載一次即可。這是 Apple 的安全機制。

目前完成度如何？

基本功能已可正常使用。伺服器本體佔用空間不到 20MB，使用 Rust 編寫；作者認為整體寫得還不錯。

我的電腦/筆電可以跑伺服器嗎？

非常有可能。只要你的平台受到 Rust 支援，理論上就可以。若沒有現成構建，建議先手動編譯專案。ew repo

伺服器現在掛了嗎？我連不上

如果你剛剛就是從目前使用的伺服器載入這個頁面，那答案通常是否定的；否則請聯絡伺服器管理員。

活動能正常運作嗎？

多數活動尚未完整實作，不過大部分應該不會讓遊戲崩潰。Star 活動已有部分實作，可以提升排名並與其他玩家競爭排行榜，但目前尚未實作活動獎勵。

那我要怎麼取得活動 UR？

有一些序號可以取得不同內容，其中包含活動 UR。具體序號內容可能會變動，建議查看 serial code 檔案以取得最新序號。

為什麼我做某件事時遊戲會崩潰？

這通常代表伺服器端某個功能還有問題。如果是自架伺服器，請回報給上游；如果你使用別人的伺服器，請聯絡伺服器管理員並請他們回報問題。
